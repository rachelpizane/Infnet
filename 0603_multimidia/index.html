Observações

A tag video é uma tag de conteudo multimidia e seus principais atributos são: src, controls, width, height, autoplay, muted, preload e loop.

Contudo, não é ideal utilizar o atributo src direto na tag video, porque se o navegador não suportar o formato do video, ele não irá renderiza-lo

Para o autoplay funcionar atualmente nos navegadores, o video deve estar com o atributo muted.

Devemos colocar mais de uma tag source porque cada navegador aceita um formato de video de diferente.

Os formatos mais comuns são: mp4, webm e ogg.

Para que o video seja responsivo, devemos colocar o atributo width="100%"

E se o navegador não suportar o video, devemos colocar uma mensagem informando que não foi possivel renderizar o video e incluir um link para baixa-lo.

O ideal é colocar atributo controls para que o usuario possa controlar o video.

A tag preload possui 3 valores: none, metadata e auto. O valor none não carrega nada, o valor metadata carrega apenas os metadados e o valor auto carrega o video inteiro.

É de bom tom colocar o atributo preload="metadata" para que o video carregue apenas os metadados
